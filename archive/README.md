# Archive

This directory keeps secondary historical material that is no longer part of the mainline result surface but is still useful for provenance.

Contents are grouped conservatively:

- `archive/reports/debug/`
  - focused debug ladders and narrow correctness investigations
- `archive/reports/milestones/`
  - superseded smoke and intermediate milestone reports
- `archive/reports/launch_prep/`
  - pre-launch planning notes that were superseded by completed runs
- `archive/logs/launchers/`
  - redundant launcher logs and stdout captures
- `archive/logs/interrupted_runs/`
  - interrupted long-run logs kept for forensics

Important:

- archived items are intentionally secondary
- the main reproducible result surface remains under:
  - `outputs/quantized/`
  - `outputs/reports/`
  - `docs/`
- some archived JSON/Markdown bundles preserve their original in-file path strings as historical records
